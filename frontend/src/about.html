About This Site

หากคุณสงสัยว่าเว็บนี้สร้างขึ้นมาทำไม เพื่ออะไร เราจะบอกให้

คือเมื่อวาน เรานั่งอ่านกระทู้ผีอยู่ ซึ่งกระทู้แนวนี้มักจะยาวมากกก

และใช้เวลาอ่านเยอะ แล้วตัวอักษรในเว็บ Pantip เองมันก้ใหญ่เหลือเกินนน เราก็ปวดตาสิ

เราก็เลยคิดว่า เออมันน่าจะดีนะถ้ามีเว็บสำหรับอ่านกระทู้พันทิพแล้วสามารถย่อขยายตัวอักษรได้

พอลองมาคิดดู เออมันก้น่าจะทำเองได้ไม่นาน ไม่น่าจะเกินหนึ่งวัน

ถึงจุดนี้บางคนอาจจะบอกว่า ใน Web Browser มันมี Reader Mode ด้วยนะ

ใช่! มันมี แต่ว่า Reader Mode เวลาใช้เปิดอ่านในเว็บ Pantip เนี่ยมันดึงมาให้แค่โพสท์หลัก มันไม่ยอมดึงคอมเม้นท์มา

เข้าเรื่องต่อ ในเมื่อเราคิดได้อย่างนั้นแล้ว จะรออะไร เราก็เลยเริ่มเขียนโค้ดเลยจ้าา

เขียนไปสามชั่วโมงก็เริ่มเป็นรูปเป็นร่าง จะยากก็ตรงจะทำให้มันออนไลน์ยังไง ไม่อยากเสียตัง

เราก้ลองโยนขึ้น Firebase ก่อนเลย แต่มันก้ดันมีปัญหาว่า Server ของระบบ Functions ที่เราใช่เนี่ยมันไปอยู่อเมริกา จะเปิดอ่านซักหน้านึง รอกันนานเลยจร้า

ในเมื่อ Firebase ใช้ไม่ได้เราก้เลยมาลอง Amazon AWS เปิดเครื่อง EC2 ที่วางอยู่ที่ Singapore แต่ก็ดูเหมือนจะมีปัญหา

EC2 ของ Singapore โดน Pantip บล็อค!

ทำไงล่ะทีนี้ ทางออกสุดท้าย โยนขึ้น Server ที่บ้านก่อนให้มันพอมีใช้ไปก่อน เดี๋ยวค่อยเปลี่ยนทีหลัง


สุดท้ายนี้หากว่าวันนีงมีนักพัฒนาซอฟท์แวร์ของ Pantip ผ่านมาเห็น อยากให้พิจารณาทำระบบช่วยอ่านขึ้นมาหน่อยนะครับ เพื่อประโยชน์สุขของชาว Pantip

ป.ล. หากมีคนเข้ามาใช้เยอะขึ้นจน Server ที่บ้านเราดึงอินเตอร์เน็ตเยอะเกินไปจนทำให้เน็ตที่บ้านเราช้า ก็เห็นทีจะต้องย้ายออกและหันมาเปิดรับเงินบริจาคสบสบทุนกันต่อไป
